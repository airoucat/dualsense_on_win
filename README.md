# DualSense 适配知识库索引

## 目标
为项目提供可维护的 DualSense / DualSense Edge（DSE）输入输出适配方案：
- 输入：按键、摇杆、扳机、触摸板、传感器
- 输出：震动、触觉反馈、自适应扳机、灯光
- 连接：USB/蓝牙
- 运行：独占/共享、冲突管理、热插拔重连

## 文档阅读顺序
1. DS-00-架构总览
2. DS-10-设备发现与打开策略
3. DS-20-输入报告解析方案
4. DS-30/31/32/33 输出能力文档
5. DS-40 USB/BT 差异
6. DS-50 DSE 兼容
7. DS-60/70 运行稳定性
8. DS-90 实施验收

## 约定
- 本目录以“思路和架构”为主，避免硬编码细节污染。
- 具体字节位定义、CRC、report 长度等，统一在后续“实测附录”补充。
- 每次确认一条协议细节，就更新对应文档“已验证项”。
